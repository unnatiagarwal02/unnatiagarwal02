**Tech:** `Python` `XGBoost` `SHAP` `MLflow` `FastAPI` `Streamlit` `AWS S3`

Built end-to-end signal quality pipeline on MIMIC-III waveform data. Engineered frequency-domain features from raw EEG/ECG time series using MNE-Python and deployed Isolation Forest anomaly detection with clinical visualization.

**Tech:** `Python` `MNE-Python` `SciPy` `Scikit-learn` `Streamlit`
